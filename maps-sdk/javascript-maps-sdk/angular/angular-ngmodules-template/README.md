# ArcGIS Maps SDK for JavaScript Angular NgModules template

📁 **[Click here to download this directory as a ZIP file](https://esri.github.io/developer-support/maps-sdk/javascript-maps-sdk/angular/angular-ngmodules-template/angular-modules-template.zip)** 📁

This Angular project uses the ArcGIS Maps SDK for JavaScript 5.x with the traditional NgModule architecture instead of standalone components.

While standalone components are now the default and recommended approach for modern Angular applications, many existing applications still use NgModules. This project provides a simple starter template for developers who need to integrate the ArcGIS Maps SDK into an Angular application built with NgModules.

If you are building a new Angular application and would prefer to use standalone components, refer to the official standalone template from [here](https://github.com/Esri/jsapi-resources/tree/main/templates/js-maps-sdk-angular)

## Get started
1. Open a terminal window in the project directory
2. Run npm install

## To test the development
1. Run npm start
2. Open http://localhost:4200/

## To build and test the build
1. Run npm build or npm watch
2. Run cd dist\angular20-modules\browser
3. Run http-server
4. Open http://192.168.1.117:8080 or http://127.0.0.1:8080

## To run unit tests
1. Run npm test
